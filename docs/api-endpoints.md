# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users/:id`
- `GET /api/users/:id`
  - User Profile

### Session

- `POST /api/session`
- `DELETE /api/session`

### Teams

- `GET /api/teams`
- `POST /api/teams`
- `GET /api/teams/:id`
- `DELETE /api/teams/:id`

### Projects

- `GET /api/projects`
  - Projects Index
- `POST /api/projects`
- `GET /api/projects/:id`
- `PATCH /api/projects/:id`
- `DELETE /api/projects/:id`

### Tasks
- `GET /api/tasks`
- `POST /api/tasks`
- `GET /api/tasks/:id`
- `DELETE /api/tasks/:id`
